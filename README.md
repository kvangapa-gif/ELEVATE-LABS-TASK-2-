To-Do List Manager – Python

This project is a simple command-line based To-Do List Manager.
It allows the user to add tasks, view all saved tasks, and remove tasks.
All tasks are stored in a text file so they remain even after closing the program.

Features
View Tasks: Displays all tasks in a clean numbered format.
Add Task: Lets the user enter a new task which is saved automatically.
Remove Task: Deletes a task by selecting its number.
Persistent Storage: Tasks are stored in tasks.txt.

Files Used
tasks.txt — Stores all the tasks line by line.

How to Run

Save the Python code in a file, for example: todo.py.

Run the program using:
python todo.py
